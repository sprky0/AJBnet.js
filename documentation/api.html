Basic Implementation

Part one, load the framework

Include the framework by giving it a script tag with a data-init attribute containing the configuration object.

<script type="text/javascript" src="ajbnet/ajbnet.js" data-init="{srcBasePath:'js/',shorthand:'MyNamespace'}"></script>

Your main function

In terminology derived from c and similar languages, your 'main' function is the body of the application, which sets everything into motion. This particular main has a dependency on a particular version of jquery, which will be loaded before this code is run.

Advanced API (Methods)

AJBnet.init(options)

The AJBnet.init method, and data-init attribute of the originating script tag both accept an object which can contain one or more of the following properties.

srcBasePath
the base path at which the library should attempt to load libraries.
shorthand
String containing a reference name to use for the global AJBnet library object. This is provided to allow the library to be nested in another namespace, or simply referred to differently (eg: YourSite.AJBnet or simply YourSite). This will be declared in window / global scope when the AJBnet.init() method is called by autoInit or manually.
debug
Boolean true or false to enable or explicitly disable logging.
app
String or Array containing the classpath(s) of the desired main function(s).
main
An alias for app

AJBnet.run(classpath)

If the dependency has been loaded, run the closure at the supplied classpath, and return its value. If that particular dependency does not exist yet, attempt to requre it (this must happen asyncronously).

AJBnet.construct(classpath,argument0,argument1, ... argumentN)

Attempt to construct a new object from the constructor returned by the classpath. This method will pass the supplied arguments to the constructor.

AJBnet.global(key,value)

Set or get value by key. If value is excluded, it will return the value set previously. If value is included, it returns a reference the AJBnet object so it can be chained.

AJBnet.ready(closure)

Run the passed closure when the document is available. This can be nested in a define block or used separately.

AJBnet.execute(closure)

Run a closure and return the returned value. If the variable passed is not a closure, this will return null.

AJBnet.log(text,type)

Try to write the text supplied to the AJBnet.logFunction() method. The AJBnet.log method checks against the enabled log types to see if the log should be written or not.
